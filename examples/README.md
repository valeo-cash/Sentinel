# Sentinel Examples

| Example | Description | Difficulty |
|---------|-------------|------------|
| [x402-agent-starter](./x402-agent-starter) | Simplest x402 agent | Beginner |
| [x402-fleet-manager](./x402-fleet-manager) | 5 agents with budgets | Intermediate |
| [x402-enterprise-compliance](./x402-enterprise-compliance) | Production compliance | Advanced |
| [x402-nextjs-saas](./x402-nextjs-saas) | SaaS with x402 payments | Intermediate |
| [x402-langchain-agent](./x402-langchain-agent) | LangChain research agent | Intermediate |

Each example is a self-contained project. Clone any one and run it independently.

## Quick Start (any example)

```bash
cd examples/<example-name>
npm install
npm start
```

All examples track x402 payments with [Sentinel](https://sentinel.valeocash.com).
