# Roadmap

## Planned

- **Multi-agent budget coordination** — Shared team budgets across agents
- **Escrow and pre-commitment flows** — Reserve budget before payment
- **Treasury management integration** — Connect to on-chain treasuries
- **On-chain agent identity verification** — Link agents to verified identities
- **Real-time dashboard** — Web UI at [app.valeo.money](https://app.valeo.money)

## API Backend

- **api.valeo.money** — Centralized audit ingestion, query API, analytics
- **PDF export** — Compliance report generation via remote API
- **Sync** — Push local records to remote for fleet-wide visibility
